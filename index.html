O que estou aprendendo

A arte da curiosidade

É impossível falar sobre o que eu estou aprendendo sem atrelar a curiosidade no meio, pois com ela estou tendo o costume de fazer perguntas diretas e perguntar o que de fato eu tenho dúvida de forma objetiva.

Desenvolvimento font-end

Algumas palavras-chave que fazem parte da minha rotina como desenvolvedor front-end: HTML, CSS, JavaScript, consumo de APIs, Node.js, GIT, tudo isso combinado para o melhor desempenho na minha jornada.

JavaScript

O que curto de fato em uma linguagem é a utilidade que ela pode proporcionar, e nisso o JavaScript não está desapontando, já conheço conceitos da linguagem como operadores lógicos e ternários, DOM, condições, funções, entre outras utilidades da linguagem.

CSS

A linguagem que cria a beleza para o seus projetos, estou aprendendo junto com HTML e JavaScript e no momento meus estudos estão focado em grids, media queries para aplicação responsiva, display, e flexbox.

HTML

Estou aprendendo HTML e toda sua história, estrutura, documentação, semântica, boas práticas para dados na web com o padrão W3C que visa compreensão, capacidade de conexão, facilidade e descoberta, confiança e facilidade de acesso.

GIT

Pausei em GIT depois de tudo isso, para aprender um pouco mais sobre o controle de versionamento de projetos e venho achando bastante útil, já vinha usando o Github para hospedar projetos mas com o GIT a produtividade aumentou pelo fato de poder manipular os projetos localmente e server side.

Linux

Junto com GIT me senti na necessidade de aprender um pouco mais sobre esse S.O, e desde então venho usando um dual boot para programar de fato no ambiente Linux e é por lá que eu faço tudo que envolva programação.

Node.js

O meu objetivo "final" é ser um desenvolvedor Fullstack, e para isso estou buscando me familiarizar com o Back-end através do Node.js onde pretendo criar aplicações multiplataforma pra não haver problemas com tipo de Sistema Operacional (S.O).

UX UI

Ainda tem a parte dedicada ao usuário que é o consumidor final, sabendo disso em UX estou aprendendo persona, pesquisa e análise de dados, e em UI estou focado em design, tipografia e colorimetria.
